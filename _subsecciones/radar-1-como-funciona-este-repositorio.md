Este es un **contenido de ejemplo**. Cuando publiquéis el primer radar real, esta noticia se sustituye desde el editor visual, sin tocar código.

Cada subsección admite:

- **Negritas**, _cursivas_ y enlaces
- Listas con viñetas o numeradas
- Imágenes y tablas
- Un PDF adjunto descargable
- Un enlace externo opcional

Cada noticia tiene **su propia dirección web fija**. Al abrirla, abajo del todo verás un recuadro con el enlace listo para pegar en Zoho Campaigns.
